# 🚚 Supply Chain Analytics Dashboard

## 📊 Power BI | SQL | Data Analytics Project

This project analyzes supply chain operations to provide insights into shipment performance, delivery efficiency, transportation modes, product movement, and operational trends.

The project combines **MySQL for data analysis** and **Power BI for interactive dashboard development** to help businesses monitor supply chain performance and identify operational inefficiencies.

---

## 📌 Project Overview

The objective of this project is to analyze supply chain data and develop an interactive dashboard that provides insights into:

- Shipment performance
- On-time delivery
- Fill rate
- Shipment volume
- Transportation modes
- Regional performance
- Warehouse performance
- Shipment trends
- Delivery delays
- Operational efficiency

The dashboard enables users to interactively analyze supply chain performance using filters such as:

- Year
- Month
- Region
- Country
- City
- Warehouse
- Transportation Mode

---

# 🛠️ Tools & Technologies Used

- **MySQL** – Data analysis and KPI calculations
- **Power BI** – Interactive dashboard development
- **Power Query** – Data cleaning and transformation
- **DAX** – KPI calculations and analytical measures
- **Excel/CSV** – Data source
- **GitHub** – Project documentation and version control
